# Rencana: Fix Voice Disconnect + DNS

## Root Cause
Bot crash karena DNS error (`EAI_AGAIN`) → unhandled error di VoiceConnection → Node.js crash → bot restart tapi voice channel hilang.

## Perubahan 1: Ganti DNS di Server
**Target:** Server `192.168.1.10`

### Steps:
1. Set DNS ke Google (8.8.8.8, 8.8.4.4) via `netplan` atau `systemd-resolved`
2. Hapus Cloudflare (1.1.1.1)
3. Restart DNS service
4. Verifikasi dengan `nslookup discord.com`

### Command:
```bash
# Cek netplan config
ls /etc/netplan/

# Edit config (backup dulu)
cp /etc/netplan/*.yaml /etc/netplan/*.yaml.bak

# Set DNS ke Google
sudo netplan apply

# Atau langsung via systemd-resolve
sudo resolvectl dns wlo1 8.8.8.8 8.8.4.4

# Verifikasi
nslookup discord.com
```

## Perubahan 2: Fix Kode Bot
**Target:** `C:\coding\Stella\Asistent AI Telegram\discord_bot.js`

### Fix 2a: Tambah error handler di VoiceConnection (line ~455)
```js
connection.on('error', (error) => {
    console.error('[Voice] Connection error:', error.message);
});
```

### Fix 2b: Tambah state change handler untuk reconnect (setelah line 455)
```js
connection.on(VoiceConnectionStatus.Disconnected, async (oldState, newState) => {
    if (newState.status === VoiceConnectionStatus.Disconnected) {
        console.log('[Voice] Disconnected, attempting reconnect...');
        // Coba reconnect beberapa kali
        for (let i = 0; i < 3; i++) {
            try {
                await entersState(connection, VoiceConnectionStatus.Ready, 10_000);
                console.log('[Voice] Reconnected successfully');
                return;
            } catch (e) {
                console.log(`[Voice] Reconnect attempt ${i + 1} failed`);
            }
        }
        console.log('[Voice] Reconnect failed, cleaning up');
        delete voiceConnections[oldState.guild.id];
        delete audioPlayers[oldState.guild.id];
    }
});
```

### Fix 2c: Tambah idle timeout (opsional)
Cek setiap 60 detik apakah bot sendirian di voice. Kalau sendirian > 5 menit, auto-leave.
